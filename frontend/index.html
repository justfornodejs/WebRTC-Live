超低延迟

基于 WebRTC 协议，端到端延迟可控制在 50-300ms，远优于 HLS/RTMP，完美适配实时同屏场景。

🔌

零插件

纯浏览器原生 API 实现，不依赖 Flash 或任何浏览器插件，主流浏览器开箱即用。

🖥️

多源推流

支持屏幕共享、摄像头、本地视频文件三种推流模式，满足多种场景需求。

📊

实时监控

推/拉流过程中实时展示分辨率、帧率、码率等统计信息，连接状态一目了然。

🐳

Docker 部署

SRS 服务器通过 Docker Compose 一键部署，开箱即用，降低运维成本。

🔒

安全传输

支持 HTTPS + DTLS 加密传输，WebRTC 媒体流自带 SRTP 加密，保障数据安全。
